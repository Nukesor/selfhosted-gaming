_Feed the Beast_ is an awesome [website](https://feed-the-beast.com/modpack) where many people create and maintain Minecraft modpacks.

New modpacks are coming out all the time and you can even still play old modpacks from 2013.
In general, it's just a very pleasant experience playing modpacks from Feed the Beast.


## Installation

1. Install Java, so you can run the `java -h` command on your command line.
    For really old modpacks, you might need to install an older java version.
    Check whether it works with a new version.
    If it doesn't, OpenJDK-8 should do the job for most old packs.
1. Go to the Feed the Beast [mod site](https://feed-the-beast.com/modpack).
1. Select your modpack.
1. Go to `Versions` and download the Server version for your operating system.
1. Fail. The new installer offer binary blobs that don't have any context on how to use them.
    They're pretty much useless in their current state.

1. Create the directory where you want your Minecraft server to live in.
    This will be referred to as `$GAME_DIR`.
1. Copy the `server.jar` into `$GAME_DIR`.
